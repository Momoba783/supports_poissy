Les balises HTML

Les sélecteurs de type ciblent les élements de la page HTML par les balises. Par exemple si nous voulons cibler tous nos titres de niveau <h4> de la page HTML, nous utiliserons le sélecteur de type h4

HTML :
<h4>Bonjour à tous</h4>
<h4>Bye tout le monde</h4>
CSS :
h4 {
...
}

les sélécteurs de classe permettent de cibler un élément en fonction de la valeur de l'attribut class attaché à cet élément. Les sélecteurs de class permettent d'avantage de précision car ils permettent de cibler les éléments d'un groupe ayant une classe identique plutôt toutes les balises HTML de la page. Les sélecteurs de classe permettent d'appliquer la même mise en forme en attribuant la même valeur dans l'attribut class à différentes balises HTML. En CSS les classes sont appelées avec un point. Ici, le sélecteur de la classe ciblera tout élément contenant la classe dont la valeur est "zone1", y inclut les éléments division et paragraphe.

Les Sélecteur d'ID sont encore plus précis que les sélecteurs de class, puisqu'ils ciblent un éléments unique. Tout comme les sélecteurs de class utilisent la valeur d'une class présente dans le HTML, les sélecteurs d'ID utilisent l'attribut ID.

▲ ▲ ▲ Attention ! Très important : peu importe la balise dans la quelle ils sont insérés, la valeur des attributs ID ne peuvent être utilisés qu'une seule fois dans un fichier HTML. C'est pourquoi les ID sont reservés pour les éléments significatifs.
En CSS, les ID sont appelés avec un dièse (ou hashtag). Ici le selecteur d'ID ciblera uniquement l'élement HTML qui possède un attribut ID dont la valeur est "zone3"
