Aide Mémoire des sélecteurs CSS déjà vus

  • *Sélectionne TOUS les éléments de la page html
  • .ma-jolie-nav séléctionne tous les éléments ayant le nom de la class "ma-jolie-nav"
  • #mon-unique-id sélectionne tous les éléments ayant le nom de mon id "mon-unique-id"
  • div, article, aside sélectionne toutes les balises <div>, <article> ET <aside>
  • nav a sélectionne les liens <a> inclus/imbriqués dans la balise <nav>
  • nav a :hover sélectionne tous les liens de la nav et active une animation lors du survol de la souris

Propriétés CSS utilisées dans cette page

Décrivez leurs usages et leurs valeurs possibles

  1. background : couleur de fond de l'élément
  2. color: couleur de la police
  3. float : effet flotant
  4. font-family : police
  5. font-size : taille de la police
  6. height : hauteur de l'élement
  7. left : position à partir de la gauche de la fenêtre
  8. margin : marges exterieurs
  9. padding : marges interieures
  10. position : position de l'élément dans la page (on a utilisé que la valeur (fixed) pour l'instant
  11. text-align: alignement du texte
  12. text-decoration: changement des valeurs héritées (ici le souligné est standard pour les liens sous tous les navigateurs est supprimé)
  13. top : position à partir du haut de la fenêtre
  14. transition : temps de transition lors d'une animation (ici sur les liens de la barre de menus)
  15. width: largeur de l'élément par rapport à la taille de la fenêtre si on indique une valeur relative (%, .em, etc)
